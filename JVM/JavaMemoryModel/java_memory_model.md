# Java 内存模型（JMM）

# 基本概念

- 程序：可执行二进制代码序列（静态概念）
- 进程：程序由磁盘载入到内存被 CPU 执行的完整过程，系统资源分配的最小单位（动态概念）
- 线程：分配 CPU 执行时间的最小单位
