# Java `ArrayDeque` 简介

ArrayDeque 是以动态数组实现的「双端队列」线性数据结构容器，不允许存放`null`值，在 JDK 1.6 版本被加入到 Java 集合框架中，是一个比较新的 Java 集合框架实现类。ArrayDeque 并不是线程安全的集合容器。
