# 图（Graph）基本概念

| 术语           | 含义      |
| :------------ | :-------- |
| 顶点（Vertex） | 图的基本单位 |
| 边（Edge）     | 图的顶点之间的关联关系 |
| 有向图（Directed Graph） | 图的边带有方向 |
| 无向图（Undirected Graph） | 图的边没有方向 |
| 有权图（Weighted Graph） | 图的边带有权值 |
| 无权图（Unweighted Graph） | 图的边没有权值 |

> 表：图的基本概念

# 图的分类

|         | 无向（Undirected） | 有向（Directed） |
| :------ | :---------------- | :------------- |
| **无权** | 无向无权图         | 有向无权图       |
| **有权** | 无向有权图         | 有向有权图       |
